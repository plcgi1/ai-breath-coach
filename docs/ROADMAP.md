## AI-тренер дыхания, который:

- подбирает дыхательные практики под задачу
- адаптируется под состояние пользователя
- работает в формате коротких сессий (2–5 минут)

### Этап 1: Проектирование и Упаковка (Дни 1–7)

Сборка «Золотого фонда» техник: 
    Отбор 10-12 базовых упражнений с научным обоснованием (для фокуса, анти-стресса, сна).

Логика AI-интерфейса: 
    Описание сценариев, как AI будет интерпретировать запросы пользователя («горю», «туплю», «бесит»).

Ценовой осязатель: 
    Окончательное утверждение цен и настройка приема платежей через белорусский эквайринг 
    (bePaid/Prodamus) для физлица на НПД.

### Этап 2: Сборка продукта (MVP) (Дни 8–17)

Интерфейс в Telegram: Создание Web App (TWA) — визуальный круг дыхания, кнопки быстрого выбора, окно чата с AI.
Сердце системы: Настройка логики подбора техник (связка твоего сервиса с ИИ-провайдером).

Первый тест: Закрытый запуск на 5–10 знакомых для проверки «не тупит ли бот» и проходят ли платежи.

### Этап 3: Органический «Взрыв» (Дни 18–24)

Публикация на VC.ru / Хабр: Статья в стиле «Как я сделал продукт для борьбы с выгоранием из Минска за 2 недели». 
Это основной источник бесплатных юзеров.

Запуск Lifetime-оффера: Открытие продаж «Пожизненного доступа» по спеццене для первых 100 человек 
    (создание дефицита).

Регистрация в каталогах: Массовое добавление бота во все бесплатные листинги и подборки нейросетей.

Этап 4: Масштабирование и Удержание (Дни 25–30)
    Виральная петля: Внедрение системы «Приведи друга — получи доступ» для снижения затрат на привлечение.

Сбор обратной связи: Корректировка цен или фичей на основе того, за что люди реально платят.

Анализ юнит-экономики: Подсчет реальной маржи и планирование бюджета на следующий месяц (уже из заработанных денег).

Итоговая цель к 30-му дню:

У тебя есть работающий Telegram-бот, база из первых 500+ бесплатных пользователей 
и первые 10–20 продаж Lifetime-тарифа, которые полностью окупают сервер, API и твои минимальные расходы.
