Average Revenue Per User (ARPU): 350 ₽

Минус эквайринг (4%): -14 ₽

Минус налог (10%): -35 ₽

Минус затраты на AI/Инфраструктуру: ~10 ₽

Маржинальность твоего продукта: ~83%. Для IT-стартапа это отличный показатель. Основная борьба развернется за стоимость привлечения клиента (CAC).
